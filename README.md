## Family rides

Criteria

- 60km / 40 miles (ish) - can ride in a day.
- Minimal roads
- Doable on hybrid bike (in Spring-Summer)
- Child-friendly elevation (< 500)
- Train station at either end.

Tools

- [Routing - brouter.de](https://brouter.de/brouter-web/)
- [NCN - OS Maps](https://explore.osmaps.com/?lat=52.74595&lon=-1.8777&zoom=7.0519&overlays=os-ncn-layer&style=Standard&type=2d)
